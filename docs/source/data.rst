================
Data Preparation
================

The Data Fetchers is a crawler module of pricing data, fundamental data and other market data like market value. Fetchers allow us to crawl all of the data we will need from `Tushare <http://tushare.org/>`_ or other sources.

Proxy Pool
~~~~~~~~~~

To get fundamental Data, we need set up a proxy pool for our :class:`~factorset.factors.Fundcrawler.Fundcrawler`.

Configuration file
~~~~~~~~~~~~~~~~~~

.. code-block:: ini
   :linenos:

   [TARGET]
   ;all = all
   ;all = hs300
   all = 000001.SZ, 000002.SZ

   [OPTIONS]
   MONGO = False
   CSV = True
   ;succ and fail list of symbol
   SFL = True

   [STORE]
   hqdir = ./hq
   ;Only fund crawler will use proxy_pool
   funddir = ./fund
   otherdir =  ./other

   [ARCTIC]
   ;host = 127.0.0.1:27017
   host = localhost

   [READ]
   proxypool = 127.0.0.1:5010
   proxymin = 5
   encode = gbk
